# Mittal Memory Log

A playful family memory-log website starter.

## What's included

- `index.html` and `index.css` for the homepage
- Separate pages for:
  - Rishabh Mittal
  - Siddhant Mittal
  - Neha Mittal
  - Akhil Mittal
- A `travel` section
- A sample trip folder with its own HTML and CSS
- An `assets/images` folder for future photos

## How to use

1. Unzip this folder.
2. Open `index.html` in your browser.
3. Replace placeholder quotes and text.
4. Add family photos inside `assets/images`.
5. For a new trip, copy `travel/sample-trip`, rename it, and edit the HTML/CSS.
6. Each family member can customize their own `family/name/style.css`.
